# Bank Management System

A Java Web-Based Bank Management System developed using JSP, Servlets, JDBC, MVC Architecture and MySQL.

## Features

- User Registration
- User Login
- Deposit Money
- Withdraw Money
- Transfer Money
- Check Balance
- Last Transactions
- View Profile
- Edit Profile
- Change Transaction PIN
- Logout
- Input Validations

## Technologies Used

- Core Java
- Advanced Java
- JSP
- Servlets
- JDBC
- MySQL
- HTML
- CSS
- Eclipse IDE
- Apache Tomcat

## Project Structure

- Controllers
- Models
- JSP Pages
- MySQL Database
